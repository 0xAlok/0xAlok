## Current Projects

- [ArchVizAI](https://github.com/0xAlok/ArchVizAI) - AI-assisted interior visualization.
- [Spatia](https://github.com/0xAlok/Spatia) - Parametric 3D room configurator for architecture/interior workflows.
- [Quiz-Agent](https://github.com/0xAlok/Quiz-Agent) - Autonomous LangGraph/LangChain agent that solves multi-step data quizzes.
- [Messy-Mashup](https://github.com/0xAlok/MessyMashup) - Deep learning project for music genre classification under noisy mixing conditions.
- [Cinema-Audience-Forecasting](https://github.com/0xAlok/Cinema-Audience-Forecasting) - Forecasting project for theatre audience prediction competition.
- [LaunchPad](https://github.com/0xAlok/LaunchPad) - Campus recruitment platform for institutes, recruiters, and students.
- [TabNest](https://github.com/0xAlok/TabNest) - Browser extension to save and restore tab sessions in one click.
- and more!
